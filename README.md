# Modern MERN Stack Developer Portfolio

A responsive, modern portfolio website for Soumyashree Nayak, showcasing MERN stack development skills with a dark theme and smooth animations.

## 🚀 Features

- **Responsive Design**: Works perfectly on desktop, tablet, and mobile devices
- **Dark Theme UI**: Modern dark theme with neon blue and purple accents
- **Smooth Animations**: Framer Motion and AOS animations throughout
- **Glassmorphism Effects**: Modern glass-like UI components
- **Interactive Elements**: Hover effects, scroll progress, and animated components
- **Professional Sections**: Hero, About, Skills, Projects, Experience, Education, Contact, Footer

## 🛠️ Technologies Used

- **React JS** - Frontend framework
- **Bootstrap 5** - CSS framework for responsive design
- **Framer Motion** - Animation library
- **AOS (Animate On Scroll)** - Scroll animations
- **React Icons** - Icon library
- **React Type Animation** - Typing effect
- **Vite** - Build tool

## 🚀 Getting Started

1. **Clone the repository**
   ```bash
   git clone <repository-url>
   cd portfolio
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start development server**
   ```bash
   npm run dev
   ```

4. **Open in browser**
   ```
   http://localhost:5173
   ```

## 📸 Images

Add your profile image as `public/profile.jpg` (recommended size: 400x400px)

Project images should be placed in `public/` directory as:
- `project1.jpg`
- `project2.jpg`
- etc.

- Secondary: `#8a2be2` (Purple)
